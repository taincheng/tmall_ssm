# tmall_ssm
ssm框架搭建模仿天猫的电商系统

1.项目的介绍
  实现了前台和后台功能的模仿天猫整站。
2.使用的技术
  存储层：Mybatis
  控制层：SpringMvc
  项目格式：Maven Web项目
  视图模板：JSP
  前端框架：Bootstrap
  异步处理技术：JQuery
  使用IDE：IDEA
  服务器：IDEA内置tomcat
 
3.后台需求
